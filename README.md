Esse código disponibilizado é um simulador da copa do mundo desde a repescagem atéa final, a simulação segue a seguinte ordem:



Repescagens europeias;

Repescagens mundiais;

Fase de grupos;

Pré-oitavas;

Oitavas de final;

Quartas de final;

Semi-Finais;

Final;

RECOMENDAÇÕES:
Utilize o codespaces disponibilizado no github seguindo as instruções abaixo:

Abra um novo espaço vazio;

Crie um arquivo com o nome copa.c e cole o código nele

Instale as biblitecas recomendadas

No canto superior da tela clique para rodar o código; 



Cuidado ao digitar um placar, caso você digite errado ele irá salvar o valor errado;
